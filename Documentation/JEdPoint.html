Introduction

JEdPoint is a completely modular, open-sourced (GPL) point program for Fidonet (http://fidonet.org). JEdPoint was written in Java so that many platforms could benefit from its existence. Modules are completely interchangable, meaning that you can change them at will and the program will still function as normal.

Before we Begin

This documentation assumes you are relatively acquainted with your system. It assumes you know what a point number is. It assumes you know how to run java.exe (or javaw.exe). It assumes you have enough common sense to figure things out for yourself.

Installation

  1. Install Java

    Windows users: Be sure to download the international version of the Java Runtime Environment (JRE).

  2. Unzip JEdPoint to a directory

    Any directory will do.

  3. Edit JEdPoint.cfg

    Change the username.
    Change the point number.

  4. run java -jar JEdPoint.jar

    Windows users: You can try to use the following shortcut to start JEdPoint

    %comspec% /c start /MAX "" "PathToJavaw.Exe" -jar JEdPoint.jar

That's it. You have the option of configuring the modules (look in the settings/ directory) but that should be unnecessary for most people.

Developers

Writing new modules for JEdPoint should be easy. Check the Documentation/Developers directory for more information on how to write modules for JEdPoint.
